# 💬 Frontend React - WhatsApp Sistema

Interface web estilo WhatsApp Web construída com React + TypeScript + Tailwind CSS.

## 🎨 Features

### Interface WhatsApp-like
- ✅ Sidebar com lista de conversas
- ✅ Preview de última mensagem
- ✅ Badges de mensagens não lidas
- ✅ Status do atendimento (Bot, Aguardando, Em atendimento, Finalizado)
- ✅ Chat em tempo real
- ✅ Bolhas de mensagem (enviadas/recebidas)
- ✅ Indicador de leitura (✓ enviada / ✓✓ lida)
- ✅ Input de mensagem com auto-resize
- ✅ Scrollbar customizado

### Funcionalidades
- ✅ Listar todas as conversas
- ✅ Filtrar por status
- ✅ Buscar conversas
- ✅ Ver detalhes do cliente
- ✅ Enviar mensagens
- ✅ Assumir atendimento
- ✅ Finalizar atendimento
- ✅ Transferir para bot
- ✅ Atualização automática (polling 5s)

## 🚀 Executar

```bash
npm start
```

Abrirá em `http://localhost:3000`

## 📊 Stack

- React 18 + TypeScript
- Tailwind CSS
- Zustand
- Axios
- React Icons
- Date-fns

╔══════════════════════════════════════════════════════════════╗
║        🚀 COMO INICIAR O SISTEMA WHATSAPP COMPLETO          ║
╚══════════════════════════════════════════════════════════════╝

📋 PRÉ-REQUISITOS:
  - Docker instalado
  - Node.js instalado
  - Python 3 instalado

═══════════════════════════════════════════════════════════════

🔵 PASSO 1: INICIAR DOCKER (PostgreSQL + Redis)

cd ~/Chat/whatsapp_system

# PostgreSQL
docker start whatsapp_postgres

# Redis  
docker start whatsapp_redis

═══════════════════════════════════════════════════════════════

🔵 PASSO 2: INICIAR BACKEND FASTAPI (Terminal 1)

cd ~/Chat/whatsapp_system/backend_fastapi
./run.sh

Aguarde mensagem:
✅ Backend rodando em http://localhost:8000

═══════════════════════════════════════════════════════════════

🔵 PASSO 3: INICIAR CELERY WORKER (Terminal 2) - OPCIONAL

cd ~/Chat/whatsapp_system/backend_fastapi
./run_celery.sh

Aguarde mensagem:
✅ Celery Worker pronto

═══════════════════════════════════════════════════════════════

🔵 PASSO 4: INICIAR FRONTEND REACT (Terminal 3)

cd ~/Chat/whatsapp_system/frontend_react
npm start

Aguarde abrir automaticamente:
✅ Frontend em http://localhost:3000

═══════════════════════════════════════════════════════════════

🎉 PRONTO! SISTEMA FUNCIONANDO!

ACESSAR:
  Frontend: http://localhost:3000
  Backend: http://localhost:8000
  Docs API: http://localhost:8000/docs

═══════════════════════════════════════════════════════════════

🧪 TESTAR:

1. Abra o frontend no navegador
2. Veja a interface WhatsApp Web
3. O WebSocket conectará automaticamente
4. Envie mensagem via webhook para ver em tempo real
